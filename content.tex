\question
If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.
Write a procedure to find the sum of all the multiples of 3 or 5 below 1000~\cite{projecteuler}.

\begin{solution}
  \begin{minted}{scheme}
(define 
  (sum35 n)
    (if (= 0 n)
      0
      (if (= 0 (modulo n 3))
      (+ n (sum35 (- n 1)))
      (if (= 0 (modulo n 5))
      (+ n (sum35 (- n 1)))
      (sum35 (- n 1))))))
  \end{minted}
\end{solution}

Define a procedure discount that takes two arguments: an itemâ€™s initial price and a percentage discount \cite{simplyscheme}.

\begin{minted}{scheme}
> (discount 10 5)
9.50
> (discount 29.90 50)
14.95
\end{minted}
  
  
\begin{solution}
  \begin{minted}{scheme}
(define (discount price reduction)
  (if (and (> reduction 0) (> 100 reduction))
  (- price
  (* (/ price 100)
  reduction))))
\end{minted}
\end{solution}
